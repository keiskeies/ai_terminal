# AI Terminal UI/UX 重设计任务计划

## 目标
为 AI Terminal 设计一套现代化、专业、跨平台一致的 UI/UX 方案，包括新 LOGO 设计、视觉系统升级和主要页面重构方案，解决当前 UI "丑" 和布局不协调的问题。

## 关键约束
- 必须保持暗色主题为主的开发者工具风格
- 必须兼容 macOS / Windows / Linux / Android / iOS
- 必须保留 SafetyGuard 的颜色语义（safe/warn/blocked）
- 终端区域保持高对比度和原生终端体验
- 设计需考虑 Flutter 实现可行性
