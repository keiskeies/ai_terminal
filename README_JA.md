## 一言で説明すると

> **ターミナルを使ったことがなくても大丈夫。** AI Terminalを開いて、やりたいことを自然言語で伝えるだけ — サーバーに接続してコマンドを実行し、ソフトウェアをインストールし、トラブルシューティングまで行います。すべて安全に、あなたの管理下で。

## 🎯 共感できますか？

### 😫 初心者 / 非技術者の方

- VPSをレンタルしたのに、ターミナルを開いたら**真っ黒な画面**で何を打てばいいかわからない
- 友達に「Nginxをインストールするだけだよ」と言われたのに、10個のチュートリアルをググってもコマンドがバラバラ
- Javaの環境構築を試したら、`PATH`を間違えてターミナルが壊れた
- サーバーの脆弱性について警告されたけど、確認方法すらわからない
- 3時間格闘したけど、何も動かない。もう嫌だ。

### 👨‍💻 開発者の方

- 毎回同じ`chmod` / `systemctl`コマンドをググっている
- サーバーにSSH接続したものの、必要な`grep`オプションを忘れてしまう
- ログを確認したい？まずは半年前のブックマークを探すところから
- ブラウザのタブが15個も開いていて、サーバーを行ったり来たりで何がどこにあるかわからなくなる

### 🔧 DevOps / システム管理者の方

- 10台のサーバーに同じソフトウェア？一台ずつSSHして繰り返し。またかよ。
- 「誰がこの設定を変更したんだ？」— 誰も覚えてない、記録もない
- 新入社員に「環境構築ってどうやるんですか？」と聞かれ、今月で5回目の説明
- 一括ヘルスチェックをしたい？スクリプトを書く方が手動でやるより時間がかかる

### 🧑‍💼 プロダクトマネージャー / 個人開発者の方

- 唯一の開発者が辞めてしまった。サーバーはブラックボックス。
- データを確認したいけどSQLが書けない。誰かに頼まなきゃ。
- 設定変更をデプロイするのに開発チームのスプリントが必要。たった1行なのに。
- 掛け持ち仕事が5つもある。`vi`を学んでいる時間なんてない。

**上記のすべてのシナリオ？AI Terminalに一言で解決できます。**

## 💡 あなたに何ができるの？

### ソフトウェアのインストール？言うだけでOK。

> 💬 「このサーバーにDockerをインストールして」

AIがOSバージョンを検出し、公式ドキュメントにマッチさせてインストールコマンドを実行し、動作確認まで行います。コマンドを覚える必要はゼロ。

### 環境設定？もうPATHの頭痛はありません。

> 💬 「Python 3.12を適切な環境変数で設定して」

AIはDebianなら`apt`、CentOSなら`yum`、macOSなら`brew`を使うことを知っています。勘ではなく、公式ドキュメントに厳密に従います。

### 脆弱性チェック？あなたよりも慎重に。

> 💬 「サーバーのセキュリティ問題をスキャンして」

AIがシステムアップデートチェック、ポートスキャン、プロセス監査を自動的に実行。修正すべき点の完全なレポートが得られます。

### ログを読む？ブックマークを掘り出す必要はもうない。

> 💬 「最近のNginxエラーを見せて」

AIはログがどこにあるか、どうフィルタリングするか、何が重要かを知っています。`tail -f`の手間なく、重要な情報だけを取得。

### サーバー管理？複数マシンを1つのインターフェースで。

SSHリモート接続にコネクションプーリングを搭載。ゼロ遅延でサーバーを切り替え。複数のタブで1つの接続を共有。

## 🛡️ セキュリティ：気になるところですよね

サーバーをAIに任せるなんて怖い。妥当な懸念が3つあります：

### 🔐 「パスワードはどこに行くの？」

```
あなたのパスワード → システムレベルの安全なストレージ（macOS Keychain / Android Keystore）
                       ↓
              ローカルデータベースには「どのキーを使ったか」だけを保存、パスワード自体は保存しない
                       ↓
              パスワードはログ、設定ファイル、ディスク上に平文で一切出現しない
```

デバイスを盗まれたとしても、生体認証/パスコードがなければ、暗号化された意味不明なデータしか得られません。

### 🤖 「AIが暴走することはないの？」

**いいえ。** 3層の防御体制：

```
┌─────────────────────────────────────────────────────┐
│ 第1層：行動境界プロンプト                              │
│ AIシステム指示が以下を明示的に禁止：                    │
│   ✗ 確認なしでのソフトウェアのインストール/アンインストール │
│   ✗ 環境変数やシステム設定の変更                        │
│   ✗ 破壊的操作の実行                                   │
│   ✓ 「確認/検査」リクエスト → 読み取り専用コマンド        │
│   ✓ 問題発見 → まず報告、自己修正は絶対にしない           │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│ 第2層：SafetyGuardコマンド分類                        │
│ すべてのコマンドは実行前にレビュー：                    │
│   🔴 blocked → 即座にブロック、実行されない              │
│      (rm -rf /, chmod 777, ディスクフォーマットなど)     │
│   🟡 warn → ポップアップ警告、CONFIRM入力が必要          │
│      (apt install, systemctl stop, ファイアウォール変更) │
│   🔵 info → 低リスクの注意喚起、通常通り実行              │
│      (curl, wget, ls, cat など)                       │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│ 第3層：あなたが最終ゲート                               │
│ あなたが常に最後の防衛線です。                          │
│ warnレベルのコマンドはCONFIRMなしでは実行されません。     │
│ いつでも中断、キャンセル、確認が可能です。                │
└─────────────────────────────────────────────────────┘
```

### 📋 エージェント行動憲章

| あなたが依頼できること | AIがすること | AIがしないこと |
|:---|:---|:---|
| ソフトウェアのインストール | 公式のインストールコマンドを生成して実行 | どのバージョンをインストールかを独自に判断 |
| セキュリティチェック | 監査コマンドを実行して結果を報告 | あなたの許可なしに問題を修正 |
| 環境設定 | 公式ドキュメントに厳密に従う | 依頼されていないシステムパラメータの変更 |
| ログ閲覧 | フィルタリングして重要情報を表示 | ログファイルの削除や変更 |
| サービス管理 | 指定されたサービスの開始/停止 | 言及されていない他のサービスの開始 |
| ワークフロー実行 | 事前定義されたステップを自動実行 | 重要なステップのスキップやプロセスの変更 |

**要するに：AIはあなたのアシスタントであり、上司ではありません。依頼されたことだけを実行します。それ以上のことはしません。**

## 🔧 AIモデルの設定

このアプリには**20以上のAIプロバイダープリセット**が付属しており、任意の**OpenAI互換API**をサポートしています：

| カテゴリ | プロバイダー |
|:---|:---|
| 🏠 ローカル | Ollama（完全無料、APIキー不要） |
| 🇨🇳 中国クラウド | DeepSeek / Qwen / GLM / Kimi / Doubao / MiMo / MiniMax / SiliconFlow / StepFun / Baichuan / Spark / Hunyuan |
| 🌍 グローバルクラウド | OpenAI / Claude / Gemini / xAI Grok / Mistral / OpenRouter / Groq |
| 🔧 カスタム | 任意のOpenAI互換APIエンドポイント |

セットアップ手順：

1. アプリを開く → 設定 → AIモデル設定
2. `+`をクリックしてモデルを追加
3. プロバイダーを選択（Base URLと推奨モデルが自動入力されます）
4. APIキーを入力し、モデルを選択
5. デフォルトモデルとして設定

> 💡 プロバイダーリストはリモート更新に対応：プロバイダードロップダウンの横にある🔄ボタンをクリックすると、サーバーから最新のプロバイダーとモデルを取得できます — アプリの更新は不要です

## 📖 デモ：ナレッジドリブン自動インストール

v1.3.0で**コマンドマニュアルナレッジベース**を導入 — 150以上の公式インストール/アンインストール/アップデートガイドを搭載。エージェントが自動的にナレッジベースにマッチさせ、公式の方法に厳密に従うことで、**AIの幻覚を排除**します。

**フローの内訳：**
